本结题报告旨在为您展示信息分析的结果，只是部分结果的展示，更多分析内容请查看结果数据文件。为保证您的阅读质量，请您仔细阅读以下说明：

1. RNA-seq

RNA-seq是基于二代测序技术的转录组学研究方法。对于small RNA的研究方法，首先提取生物样品的total RNA，利用small RNA的3'及5’端特殊结构直接在small RNA两端加上接头，然后反转录为c-DNA并进行PCR扩增，再通过凝胶电泳分离目的片段，随后进行二代高通量测序。在此基础上对small RNA序列进行参考基因比对并进行分类注释，着重对miRNA进行分析并对差异表达的miRNA进行靶基因预测，针对预测的靶基因可进行GO/KEGG富集分析，便于研究处理方法对研究物种生物功能产生何种影响。

I. 文件目录列表：为本项目的结果文件夹列表，可通过其中的链接点击打开相应的结果文件夹，方便您快捷定位相应分析结果；

II. 软件列表：展示了本项目分析时采用的软件及其版本和参数，便于您后期的处理和文章发表；

III. Method英文版：为您提供了各项分析内容方法的英文版说明，以方便您文章的撰写；

IV. 结果文件使用说明：为您提供了不同类型结果文件的打开方式说明，便于您对结果文件的解读。
